US Data Cleaning and Analysis - SQL

A project focused on cleaning and exploring a large US dataset using SQL
Key insights were derived through statistical analysis and data visualization techniques, providing a foundation for data-driven decision-making

Dynamic Sales Analysis

The goal of this project was to explore sales trends across Spanish cities and understand how factors like seasons and promotions influence revenue using Excel, Python and Power BI

Using a generic dataset, I analyzed patterns such as how summer leads to the highest sales and promotions like "20% OFF" drive the most purchases
I designed an interactive Power BI dashboard to make these insights clear and actionable

AirBnB Analysis - Tableau dashboard

An interactive Tableau dashboard analyzing Airbnb listings across various cities
This visualization highlights trends in pricing, availability, and reviews, offering insights into the short-term rental market

Project provided actionable insights for optimizing pricing strategies and understanding market dynamics across various locations, helping hosts make data-informed decisions

HR Data Analysis - Power BI

This project focused on analyzing employee hiring, promotions, and performance trends to support data-driven decision-making in HR

Using Power BI, I designed an interactive dashboard that highlights key insights, such as gender distribution in promotions, average time in roles before promotion, and hiring trends by job level

These insights enable HR teams to identify areas for improvement in diversity and career development strategies

Amazon Web Scraper with Python

This project involves building a Python-based web scraper to gather product data from Amazon, including prices, ratings, and availability
The scraper automates data collection, allowing for efficient tracking of market trends and product information
